# アプリ設計書

## 概要

メイン画面から各機能ページに遷移し、その画面が持つ機能を実行するWebアプリ

## 使用技術

- TypeScript
- Vite
- React
- HTML
- css

検討中
- Next.js

## 画面一覧

### 初期画面

- 時刻を表示する
- ログイン画面へ遷移する

### ログイン画面

- ユーザ名・パスワードを入力できる
- ログインボタンを押下することで、ダッシュボード画面へ遷移する

### ダッシュボード画面

- 各機能のページへ遷移するボタンを配置する
- 画面右上にユーザ名とIDを記載する
